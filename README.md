# How to set up

1. git clone `git@github.com:ABA-aadarsh/journal.today.git`
2. npm i
3. get tiny mce api key : reference = [Blog](https://www.tiny.cloud/blog/set-up-tinymce-api-key-in-two-minutes/)
4. register domain in mce : reference =  [Blog](https://www.tiny.cloud/blog/this-domain-is-not-registered-with-tiny-cloud/)
5. create mongodb db : reference = [Blog](https://www.mongodb.com/resources/products/fundamentals/create-database)
6. create `.env` by taking reference of `.env.example`
7. deploy to vercel
